# Ticket API Jakarta/Java EE8

Jakarta EE 8 Sample

## What's it?

It's a RESTfull API project to help you to bootstrap a new Jakarta/Java EE8 application

## Tech

- JDK 11
- Maven 3.6

#### Server

- Wildfly 19 - JEE8

#### Database

- H2(ExampleDS inside Wildfly)

#### Development

- JAX-RS 2.1
- CDI 2.0
